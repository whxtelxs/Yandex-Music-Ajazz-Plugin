# Yandex Music Ajazz Plugin
![Yandex Music Ajazz](static/git1.png)

Плагин для управления приложением Яндекс.Музыка с помощью Ajazz AKP 153 и других устройств StreamDock.

## Возможности

### Кнопки
- **Плей/Пауза** - воспроизведение и пауза трека
- **Следующий/Предыдущий трек** - переключение треков
- **Лайк/Дизлайк** - оценка текущего трека с отображением состояния
- **Случайный порядок** - включение и выключение shuffle с отображением состояния на кнопке
- **Повтор** - переключение режима повтора (выкл → список → один трек), иконка соответствует текущему режиму
- **Громкость + / Громкость −** - шаг изменения громкости настраивается в панели настроек (1–99%, по умолчанию 5%)
- **Выключить звук** - mute/unmute с отображением состояния
- **Запуск Yandex Music** - запускает Яндекс Музыку с портом отладки; при занятом порте автоматически подбирается ближайший свободный
- **Обложка трека** - обложка текущего трека; нажатие - плей или пауза
- **Название трека** - бегущая строка формата «Артист - Название»
- **Время трека** - текущее время / общая длительность
- **Discord** - отображение играющего трека в Discord с ссылкой на трек

### Энкодеры (крутилки)
- **Громкость (энкодер)** - вращение: изменение громкости, нажатие: mute/unmute
- **Перемотка (энкодер)** - вращение: перемотка трека, нажатие: плей/пауза
- **Треки (энкодер)** - вращение: следующий/предыдущий трек, нажатие: плей/пауза

### Панель настроек

В Property Inspector любой кнопки плагина нажмите **«Открыть настройки»** - откроется веб-панель с разделами:

| Раздел | Описание |
|--------|----------|
| **Соединение** | Порт отладки, запуск Яндекс Музыки, проверка соединения, отображение активного порта |
| **Громкость** | Шаг изменения громкости для кнопок и энкодера |
| **Текст** | Длина бегущей строки, размер шрифта названия и времени трека |
| **Discord** | Включение Rich Presence - текущий трек в статусе Discord |
| **Дебаг** | Режим отладки и просмотр логов плагина в реальном времени |
| **Обновления** | Проверка новых версий на GitHub, ссылка на релиз |
| **GitHub** | Сообщить о баге, предложить улучшение, поставить звезду репозиторию |

### Discord Rich Presence

При включении в настройках плагин показывает в Discord текущий трек и исполнителя. Если статус не появляется - перезапустите Discord (`Ctrl+R`) и включите **Настройки → Конфиденциальность → Показывать текущую активность**.

## Установка

Скачайте последний релиз на [GitHub Releases](https://github.com/whxtelxs/Yandex-Music-Ajazz-Plugin/releases).

<details>
<summary>Windows</summary>

1. Распакуйте папку плагина по пути:
   ```
   C:\Users\USERNAME\AppData\Roaming\HotSpot\StreamDock\plugins
   ```
   ![Yandex Music Ajazz](static/git-win-1.jpg)

2. Запустите Яндекс Музыку одним из способов:
   - **Через плагин** - добавьте кнопку **«Запуск Yandex Music»** или нажмите **«Запустить Yandex Music»** в настройках плагина
   - **Вручную** - создайте ярлык и укажите в параметрах:
     ```
     --remote-debugging-port=9222
     ```
   ![Yandex Music Ajazz](static/git-win-2.jpg)

3. Откройте StreamDock и настройте кнопки

![Yandex Music Ajazz](static/git-settings.jpg)
</details>

<details>
<summary>MacOS</summary>

1. Распакуйте папку плагина по пути:
   ```
   Библиотека → Application Support → HotSpot → StreamDock → plugins
   ```
   ![Yandex Music Ajazz](static/git-mac-1.jpg)

2. Запустите Яндекс Музыку одним из способов:
   - **Через плагин** - кнопка **«Запуск Yandex Music»** или запуск из настроек
   - **Вручную** - в терминале:
     ```
     open -a /Applications/Яндекс\ Музыка.app --args --remote-debugging-port=9222
     ```
   ![Yandex Music Ajazz](static/git-mac-2.jpg)

3. Откройте StreamDock и настройте кнопки

![Yandex Music Ajazz](static/git-settings.jpg)
</details>

## Настройка энкодеров

Для устройств с крутилками (например, Ajazz AKP05):

1. Откройте StreamDock и перейдите во вкладку **Knob**
2. Найдите категорию **Yandex Music**
3. Перетащите нужный энкодер на крутилку устройства:
   - **Громкость (энкодер)** - управление громкостью
   - **Перемотка (энкодер)** - перемотка и плей/пауза
   - **Треки (энкодер)** - переключение треков и плей/пауза

## Проверка соединения

Проверьте статус соединения одним из способов:

- В **настройках плагина** → раздел **Соединение** → **Проверить соединение**

![Yandex Music Ajazz](static/git-status.jpg)

> ⚠️ Если порт `9222` занят, плагин может запустить Яндекс Музыку на другом порту - актуальный порт отображается в настройках. Убедитесь, что приложение запущено с параметром `--remote-debugging-port`.

## Разработка и сборка релиза

### Требования
- Node.js 20+
- npm

### Установка зависимостей

```bash
cd plugin
npm install
```

### Сборка CSS

Пересобирает стили панели настроек из `propertyInspector/tailwind.input.css`:

```bash
npm run css:build
```

### Тесты

```bash
npm test
```

### Сборка релиза

1. Укажите версию в `manifest.json` (поле `Version`, формат `X.Y.Z`)
2. Соберите архив:

```bash
cd plugin
npm run prod
```

Скрипт `prod` собирает CSS, копирует проект в `release/` (без dev-зависимостей, тестов и служебных файлов) и создаёт ZIP-архив:

```
release/Yandex-Music-Ajazz-Plugin/          - папка для установки
release/YandexMusic.Ajazz.Plugin.vX.Y.Z.zip - архив для GitHub Release
```

Готовый ZIP можно прикрепить к [релизу на GitHub](https://github.com/whxtelxs/Yandex-Music-Ajazz-Plugin/releases/new).
